ZPan can be self-hosted with the Community edition. Pro and Business licenses unlock commercial capabilities on the same deployment; buying a license does not move your files to a hosted ZPan service.

![The licensing comparison in the admin console](/images/docs/admin-licensing.png)

## Choose an edition

**Community** covers core file management, sharing, image hosting, invitations, and a limited number of providers, teams, storages, and downloaders. It is suitable for personal use and small private groups.

**Pro** is intended for larger private deployments. It removes common resource limits and adds features such as custom branding, open registration, and audit logs.

**Business** is for operating storage as a service. It includes the Pro capabilities and business controls such as selling storage and traffic entitlements and site announcements.

The in-product comparison is the authority for the installed release because features can change between versions. See [Pricing](/pricing/) for the current license price and purchase terms.

A license is bound from **Admin Console → Licensing**. Keep the instance identifier and purchase account available when moving a deployment. Back up the database before migration; replacing the database may also replace the local binding state.
