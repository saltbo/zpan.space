---
title: File sharing
description: Publish files and folders with controlled public links.
---

ZPan provides two types of share links:

- **Landing shares** open a public page with file information and a download action.
- **Direct shares** redirect directly to the downloadable object.

## Create a share

Select a file or folder in the Files page, choose **Share**, and select the link type. You can optionally add:

- A password
- An expiration time
- A download limit
- Recipient notifications

Landing links are the best default for people. Direct links are useful for scripts, releases, and embeds.

## Revoke access

Open the Shares page to review active links, inspect usage, or revoke a share immediately. Revoking the share does not delete the underlying file.
