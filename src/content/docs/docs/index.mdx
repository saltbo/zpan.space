## What ZPan is

ZPan is a focused web drive for S3-compatible object storage. The server handles authentication, metadata, permissions, teams, shares, and signing. Files upload directly from the browser to object storage through presigned URLs.

It is designed for self-hosters, developers, bloggers, small teams, and automated workflows that want control over their storage without operating a traditional file-transfer server.
