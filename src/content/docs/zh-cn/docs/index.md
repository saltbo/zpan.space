## 从这里开始

ZPan 是一个专注于 S3 兼容对象存储的 Web 文件平台。服务端管理身份、元数据、权限、团队、分享和签名；文件通过预签名 URL 从浏览器直接上传到对象存储。

- 使用 **Cloudflare Workers**，无需维护服务器。
- 使用 **Docker**，部署在自己的主机和网络中。
- 连接 PicGo、PicList、uPic、ShareX 或 Flameshot，搭建自己的图床。
- 创建带密码、有效期和次数限制的文件分享。
