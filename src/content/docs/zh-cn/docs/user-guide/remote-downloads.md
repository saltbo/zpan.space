---
title: "创建远程下载"
description: 提交远程 URL、磁力链接或种子，选择保存位置，并排查排队或失败的下载任务。
---

远程下载通过管理员运行的下载节点获取 URL、Magnet 或 Torrent，再把完成的文件导入所选 ZPan 空间和目录。

![ZPan 英文版远程下载页面](/images/docs/remote-downloads.png)

在 **远程下载** 中创建任务，填写来源并选择目标；能够获取大小时先核对预计容量。目标空间必须有足够配额，节点在下载期间也需要临时磁盘。

任务会经历排队、下载、导入、完成、失败或取消等状态。排队可能只是在等待在线且兼容的节点。失败后先读错误再重试：来源不可达、节点磁盘不足、协议不支持、目标权限不足和配额耗尽需要不同处理。

取消会停止后续工作，但临时文件或保种数据可能按节点策略稍后清理。不要提交包含长期有效凭据的私有 URL，能访问任务的人可能看到来源。

完成后打开目标目录并抽样下载。所有任务持续排队或提示节点错误时，联系站点管理员检查下载节点。
