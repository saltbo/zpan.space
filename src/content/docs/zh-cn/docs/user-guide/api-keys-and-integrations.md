---
title: "API Key 与工具集成"
description: 为工具、脚本和设备分别创建、保存、测试、轮换和撤销 ZPan API Key。
---

API Key 让工具无需保存 ZPan 登录密码即可操作。应从账号设置或对应功能的集成页面创建。

![ZPan 英文版 API Key 管理页面](/images/docs/api-keys.png)

每个 Key 用一个设备和用途命名，例如 `MacBook PicGo` 或 `backup-script`。明文通常只显示一次，应立即复制到系统钥匙串、CI Secret 或密码管理器。

只授予集成所需的访问范围。不要把 Key 放进公开仓库、浏览器 URL、截图、共享配置模板，或会交付给其他用户的前端代码。不同设备使用不同 Key，丢失设备时才能单独撤销。

连接上传工具或脚本时，先核对 ZPan Base URL，再执行一次可删除的测试操作，确认返回文件、所属空间、路径和公开/私有行为。API 操作同样受限流、配额和版本权益限制。

定期撤销不用的 Key；Key 出现在日志或聊天后立即轮换。撤销只能阻止未来请求，无法收回已经下载的文件或已经传播的公开 URL。工具提示未授权时，先检查站点地址和复制时是否带入隐藏空格。
